# 60개 NFC 태그 대량 등록 프로그램

## 📋 개요
60개 락커에 NFC 태그를 순차적으로 자동 등록하는 프로그램입니다.

## 🎯 등록 순서
1. **S01-S10** (10개, 직원 구역)
2. **M01-M40** (40개, 남성 구역)  
3. **F01-F10** (10개, 여성 구역)

총 **60개 락커** 순차 등록

## 🚀 사용법

### 1. 프로그램 실행
```bash
cd ~/Desktop
python3 bulk_nfc_register.py
```

### 2. 등록 과정
1. 현재 등록 상태 확인
2. Enter 키로 시작 확인
3. ESP32에 NFC 태그를 **순서대로** 가져다 대기
4. 자동 등록 및 다음 락커로 진행
5. 60개 완료 시 자동 종료

### 3. 중단 방법
- **Ctrl+C**: 언제든지 중단 가능
- 재시작 시 중단된 지점부터 계속

## ✅ 주요 기능

### 자동 중복 방지
- 이미 등록된 UID는 자동 건너뜀
- 이미 등록된 락커는 자동 스킵

### 실시간 진행률 표시
- 현재 락커: M15 [35/60]
- 진행률: 35/60 (58.3%)
- 등록: 32개 | 건너뜀: 3개

### 최종 결과 리포트
```
STAFF   :  9/10 ( 90.0%)
MALE    : 38/40 ( 95.0%) 
FEMALE  : 10/10 (100.0%)
--------------------------------
전체    : 57/60 ( 95.0%)

⚠️ 미등록 락커 (3개):
   직원: S03
   남성: M12, M25
```

## 🔧 문제 해결

### ESP32 연결 안 됨
1. USB 케이블 확인
2. ESP32 전원 확인  
3. 다른 USB 포트 시도

### 태그 인식 안 됨
1. NFC 태그를 ESP32에 더 가까이 대기
2. 태그를 1-2초 유지
3. 다른 태그 시도

### 프로그램 멈춤
1. Ctrl+C로 중단 후 재시작
2. 재시작 시 중단된 지점부터 계속

1. **순서 준수**: 반드시 S01→S10→M01→M40→F01→F10 순서로
2. **한 번에 하나**: 태그를 하나씩만 대세요
3. **백업**: 시작 전 데이터베이스 백업 권장
